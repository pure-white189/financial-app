# 财务管家 - Android个人财务管理应用

## 📱 项目介绍
一款基于Kotlin和Jetpack Compose的现代化Android财务管理应用

## ✨ 核心功能
- 消费记录管理（添加/编辑/删除）
- 智能预算管理（月度预算、超支提醒）
- 数据分析与可视化（7天趋势图、类别排行）
- 常驻通知（创新功能，实时显示预算进度）
- 完整的深色模式支持

## 🛠️ 技术栈
- Kotlin
- Jetpack Compose + Material Design 3
- MVVM架构
- Room数据库
- Vico图表库


## 🚀 技术亮点
- iOS风格的滚动时间选择器
- 常驻通知栏预算提醒（大多数记账应用没有）
- 完整的MVVM架构实现
- 模板快速记账功能
